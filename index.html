jCookie

jCookie - a jQuery-Plugin providing an convenient api for CRUD-related cookie handling

Example

  • Create & Update: jQuery.jCookie('cookie','value');
  • Delete: jQuery.jCookie('cookie',null);
  • Read: jQuery.jCookie('cookie');
